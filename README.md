# reflexesGame

9マスのどうぶつを追いかける、ネオンアーケード調の反射神経ゲームです。  
30秒でできるだけ多くのターゲットをタップしてスコアを伸ばします。

## 仕様

- 3x3 の 9 マスにどうぶつが表示される
- 1 プレイ 30 秒で終了
- 残り時間を常時表示（`TIME xx`）
- 正解タップでスコア +1
- ミスしても減点・ゲームオーバーなし
- 成功で少しずつ速くなり、失敗すると少し緩和
- スコア `10` ごとにレベルが上がる
- レベル `1-2` は 1 匹表示、レベル `3-4` は 2 匹表示
- レベル `3-4` では、1 匹押すとその 1 匹だけ消えて `+1`、もう 1 匹は残る
- スコアは現在値のみ表示し、HUD はネオンアーケード風の発光表示にする
- 音は初期 `OFF` で、ボタンで `ON/OFF` 切替
- 縦向きでのプレイを前提（横向き時は案内を表示）
- Phaser の図形と絵文字だけで、ネオン盤面、波紋、放射線、警告点滅、レベルアップ演出を付けている

## GitHub Pages で公開する手順

1. このディレクトリを GitHub リポジトリに push
2. GitHub の `Settings` -> `Pages` を開く
3. `Build and deployment` で以下を選択
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`（または公開したいブランチ）/ `/(root)`
4. 保存後、発行された URL にアクセス

静的ファイルのみで構成しているため、追加ビルドなしで公開できます。

## 更新時の最小チェック（再発防止）

縦横案内やゲーム進行の崩れを防ぐため、変更後に次を確認してください。

1. 縦向き相当サイズ（例: `390x844`, `768x1024`）で案内が非表示であること
2. 横向き相当サイズ（例: `1024x768`）で、タッチ端末判定時のみ案内が表示されること
3. 案内はゲーム入力をブロックしないこと（非モーダル表示）
4. ゲーム開始後、残り時間が減少し 0 秒で入力受付が止まること
5. `スコア 9 -> 10`, `19 -> 20`, `29 -> 30` でレベル表示と難易度が更新されること
6. レベル `3-4` で 2 匹表示され、1 匹押したあとにもう 1 匹が残ること
7. `残り5秒` で `TIME` が警告色に変わり、視認できること
8. `tmp/ui-check/` のスクリーンショットで、HUD・盤面・下部ボタンに重なりやはみ出しがないこと
