# reflexesGame Kids Japanese Copy Design

## Goal

ネオンアーケードの見た目は維持したまま、英語中心の UI 文言を 3 歳向けのやさしい日本語へ置き換える。

## Worldview Dictionary

- ひかったら さわる
- てんすう
- じかん
- レベル
- スタート
- おしまい
- もういっかい

難しい SF 用語や英語は避け、画面の派手さは視覚とエフェクトで支える。

## Copy Direction

- 方向性: 直球キッズアーケード
- 理由: 3 歳向けでは比喩よりも `見えたもの` と `行動` が直結した文言の方が伝わりやすい
- ルール:
  - ひらがな中心
  - 短文中心
  - ネガティブ表現は弱める
  - ボタンと HUD と状態表示を同じトーンにそろえる

## Copy Mapping

- `BREACH LEVEL` -> `てんすう`
- `UPTIME` -> `じかん`
- `SYSTEM STANDBY...` -> `じゅんびちゅう...`
- `Await [ INITIATE ]` -> `スタートを おしてね`
- `[ INITIATE ]` -> `スタート`
- `[ AUDIO OFF ]` -> `おと: なし`
- `[ AUDIO ON ]` -> `おと: あり`
- `SECTOR 1` -> `レベル 1`
- `BREACH WINDOW OPEN` -> `ひかったら タッチ!`
- `NODE BREACHED` -> `ピカッ!`
- `TRACE CLEAR` -> `やった!`
- `LINK CLEAN` -> `いいぞ!`
- `TRACE LOST` -> `おしい!`
- `SIGNAL DROP` -> `あれれ?`
- `MISS` -> `ざんねん!`
- `LINK ESTABLISHED.` -> `スタート!`
- `SECURITY OVERDRIVE!` -> `レベルアップ!`
- `CONNECTION LOST` -> `おしまい!`
- `CONNECTION TERMINATED.` -> `おしまい!`
- `FINAL BREACH LEVEL` -> `きみの てんすう`
- `[ REBOOT ]` -> `もういっかい`

## Non-goals

- ターゲット形状や配色の再設計
- ルール変更や難易度調整
- 幼児向けに合わせたゲームメカニクス変更
