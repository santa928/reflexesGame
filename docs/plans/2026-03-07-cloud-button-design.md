# どうぶつ雲ボタン デザイン

**対象:** `どうぶつ かくれんぼ` の下部操作ボタン（`スタート/もういちど`, `おと ON/OFF`）

## 背景

- 現状のボタンは単色の長方形で、世界観に対して少し硬い。
- 今回は「どうぶつ感」を優先し、3歳向けに押したくなる見た目へ寄せる。
- 実装は Phaser の `Graphics`, `Text`, `Container` のみで完結させ、画像アセットは追加しない。

## 採用案

雲のようなもこもこ輪郭を持つ白基調のボタンを採用する。

- 前面: 白い雲シルエット
- 背面: パステルブルーの影レイヤーを下にずらして厚みを出す
- アイコン: 左ボタンは `🐾`、右ボタンは `🎵` / `🔇`
- ラベル色: 濃いブラウン
- 押下時: 縮小に加えて前面だけが少し沈み込む

## 形状ルール

- ベースは横長の角丸矩形
- 上部に大きめの円を 3 個、下部に小さめの円を 3 個重ねて雲のシルエットを作る
- 影レイヤーは前面と同じ形を `+8px` 下にずらして配置する
- 小さい viewport でも形の比率が崩れないよう、論理サイズで描いて `Container` 全体をスケールする

## 状態

- 通常: 前面が影の上に浮いている
- ホバー: 少し持ち上がり、ハイライトを強める
- 押下: `scale 0.95`、前面レイヤーだけ `+6px` 下へ移動
- フォーカス: ふわっとした輪郭ハイライトを表示

## レスポンシブ

- 論理サイズは `280x94`
- `768x1024` では等倍付近
- `576x768` では既存どおり `0.72` までのスケール縮小で対応

## 実装メモ

- 雲シルエットの描画仕様は純粋関数に切り出し、Node の組み込みテストで検証可能にする
- `createButton()` は雲ボタンの `Container` を組み立てる責務に寄せる
- 既存のレイアウト計算は維持し、見た目の責務のみ差し替える
