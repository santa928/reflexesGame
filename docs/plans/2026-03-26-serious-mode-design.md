# reflexesGame Serious Mode Design

## Goal

`ぴかぴかタッチ` に `しんけんモード` を追加し、通常ルールは保ったまま `誤タップ` と `ターゲット見逃し` に減点を導入する。

## Requirement Ledger

| ID | Status | Requirement |
| --- | --- | --- |
| REQ-001 | 維持 | 3x3 盤面、ネオン世界観、やさしい日本語コピー、30秒制の反射神経ゲームであること |
| REQ-002 | 追加 | ホーム画面で `ふつう` と `しんけん` を選べること |
| REQ-003 | 追加 | `しんけんモード` では誤タップと見逃しのたびにスコアを `-1` すること |
| REQ-004 | 追加 | `ふつうモード` では従来どおり誤タップ・見逃しで減点しないこと |
| REQ-005 | 追加 | 選択中モードがホーム画面で一目で分かり、開始後も内部状態として保持されること |
| REQ-006 | 追加 | スコアは `0` 未満に落とさず、表示文言は現在の操作可能性と矛盾しないこと |

## Chosen Direction

Gemini 相談は `429 MODEL_CAPACITY_EXHAUSTED` で取得できなかったため、既存のネオンUIを崩さない手動設計で進める。

- ホームカード内に `モード` ラベル、2択ボタン、選択中説明文を追加する
- `ふつう` は既存のシアン系、`しんけん` は警告色寄りのピンク系で差別化する
- プレイ中の盤面/HUD構成は増やしすぎず、開始前の選択と内部状態保持で分かる設計に留める
- 減点時は警告色フィードバックと `-1` を含む文言で即時に伝える

## State Model

- `selectedMode`: `normal` または `serious`
- `home`: モード選択と `あそぶ！` を表示する初期状態
- `playing`: 選択済みモードでゲームが進行する状態
- `paused`: ポーズメニュー表示中。`selectedMode` は保持する
- `finished`: 終了カードを表示する状態。`selectedMode` は次回開始まで保持する

## UI Spec

### Home

- タイトル: `ぴかぴかタッチ`
- サブコピー:
  - `ふつう`: `ミスでも げんてん なし`
  - `しんけん`: `ミスや みのがしで げんてん`
- モードボタン:
  - 左: `ふつう`
  - 右: `しんけん`
- 選択中ボタンは発光と彩度を上げ、非選択は抑えめにする

### Play

- 盤面/HUD/ポーズ配置は維持する
- `しんけん` の減点時のみ `-1` を含む警告文言を出す
- `ふつう` の誤タップは従来どおりスコア変動なし

### Finish

- 終了カードは既存構成を保つ
- 次の `もういちど` でも直前に選んだモードを維持する

## Responsive Rules

- `390x844`: モードボタンはホームカード幅に収まる2列で、CTA と干渉しない
- `768x1024`: ホームカード幅だけ広げ、ボタン自体は過度に大型化しない
- どちらも `タイトル -> 説明 -> モード選択 -> CTA` の順で視線が落ちるようにする

## Acceptance Criteria

- [ ] ホーム画面で `ふつう / しんけん` を切り替えられる
- [ ] `しんけん` で誤タップ時にスコアが 1 減る
- [ ] `しんけん` で見逃し時にスコアが 1 減る
- [ ] `ふつう` では誤タップ・見逃しでスコアが減らない
- [ ] スコアが `0` 未満にならない
- [ ] `390x844` と `768x1024` でホームUIが重ならない

## Non-goals

- 盤面サイズやターゲット出現ロジックの再設計
- 音色やエフェクト全体の刷新
- スコア保存やランキング追加

## Risks And Mitigations

- ホームUIが縦方向に詰まりすぎるリスク:
  カード高さとボタン縮尺を viewport ごとに制御する
- `しんけん` の減点が既存レベル制御と干渉するリスク:
  スコア下限を `0` に固定し、Tier 計算を負数に渡さない
- 文言が現在モードとズレるリスク:
  選択中モードの説明文を helper に集約して、ホーム表示とゲームロジックの参照元を揃える

## Performance Target

- 追加ボタンとテキストを増やしても、既存の描画・入力レスポンスを体感で劣化させない
