# reflexesGame Home And Pause Menu Design

## Goal

スマホ下部の誤タップを避けるため、下部の `スタート` / `おと` ボタンを廃止し、ホーム画面と右上ポーズメニューへ導線を整理する。

## Requirement Ledger

| ID | Status | Requirement |
| --- | --- | --- |
| REQ-001 | 維持 | 3x3 盤面、上部 HUD、ネオン世界観、やさしい日本語コピーは維持する |
| REQ-002 | 追加 | 初期表示はホーム画面にし、中央の大きな `あそぶ！` ボタンからゲームを始める |
| REQ-003 | 追加 | プレイ画面では右上に小さなポーズボタンを置き、タップでメニューを開ける |
| REQ-004 | 追加 | ポーズメニューから `つづける` `おと: あり/なし` `おうちへ` を選べる |
| REQ-005 | 追加 | ポーズメニューを開いている間はタイマー進行と盤面タップを止める |
| REQ-006 | 追加 | `おうちへ` ではスコア・タイマー・ターゲットを初期化してホームへ戻る |

## Chosen Direction

Gemini 相談結果のうち、`ソリッド・ネオンブロック` を採用する。

- ホームは中央寄せのタイトルカード + 大きな `あそぶ！` ボタンで構成する
- ゲーム画面からは下部操作をなくし、盤面下をデッドゾーンとして残す
- メニューは中央カードで開き、盤面より高い depth に置く
- ポーズボタンは右上固定 + 広めの hit area を持たせ、見た目より押しやすくする

## State Model

- `home`: タイトルと `あそぶ！` を表示する初期状態
- `playing`: 盤面・HUD・右上ポーズを表示し、ゲームが進行する状態
- `paused`: `playing` からメニューを開いた状態。タイマーと盤面入力を停止する
- `finished`: 終了カードを表示する状態。`もういちど` で再開し、右上ポーズから `おうちへ` も選べる

## UI Spec

### Home

- タイトル: `ぴかぴかタッチ`
- サブコピー: `ひかったら タッチ!`
- CTA: `あそぶ！`
- レイアウト: 盤面は表示せず、中央に縦積みカードを置く

### Play

- HUD は既存どおり上部中央
- 右上ポーズボタンはセーフエリア起点で `top/right` から固定距離を取る
- 盤面は中央アンカーを維持する
- 下部には操作ボタンを置かず、誤タップ吸収用の空き領域にする

### Pause Menu

- 半透明オーバーレイ + 中央カード
- ボタン順:
  1. `つづける`
  2. `おと: あり` / `おと: なし`
  3. `おうちへ`
- 背後の盤面は見せるが、入力は通さない

### Finished

- 終了カードに `おしまい!` と `きみの てんすう`
- 中央 CTA は `もういちど`
- 右上ポーズから `おうちへ` と音切替も引き続き可能

## Responsive Rules

- `390x844`: ポーズボタンの見た目は小さく保ちつつ、hit area は最低 `60x60` を確保する
- `768x1024`: 盤面は max width を守り、余白だけ広げる
- どちらも HUD / ポーズ / 盤面のアンカーを優先し、高さ割合では配置しない

## Non-goals

- 盤面サイズや難易度ロジックの変更
- PWA や service worker の変更
- 音色そのものの再設計

## Risks And Mitigations

- ポーズ中も countdown が進むリスク:
  `paused` 中は残り時間計算を止め、メニューを閉じた時だけ再開時刻を再計算する
- 盤面入力が透過するリスク:
  メニューオーバーレイとモーダル背景に interactive hit area を持たせて背面タップを防ぐ
- ホーム復帰で状態が残るリスク:
  `goHome()` でタイマー・ターゲット・スコア・表示文言をまとめて初期化する

## Acceptance Checklist

- [ ] `ホーム -> あそぶ！ -> プレイ` が成立する
- [ ] プレイ中に右上ポーズから `つづける` `おと` `おうちへ` が使える
- [ ] ポーズ中は盤面タップと時間進行が止まる
- [ ] `finished` から `もういちど` と `おうちへ` の両方が使える
- [ ] `390x844` と `768x1024` で HUD / 盤面 / ポーズが重ならない
